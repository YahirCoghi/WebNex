# Repositioning WebNex

Fecha: 2026-06-12

## Posicionamiento anterior

WebNex comunicaba principalmente servicios de paginas web: auditoria, web estrategica, web con analitica y plataforma web empresarial. La narrativa estaba centrada en presencia profesional, diseno y conversion de un sitio.

## Nuevo posicionamiento

NexSystems comunica web, software y automatizacion para empresas que quieren crecer con mas orden. La web sigue siendo una linea importante, pero ahora aparece como parte de un ecosistema digital que tambien puede incluir sistemas internos, portales, integraciones, automatizaciones y analitica.

Mensaje base:

> Web, software y automatizacion para empresas que quieren crecer.

## Audiencia

- Empresas pequenas y medianas que ya venden o atienden clientes, pero operan con procesos dispersos.
- Negocios que reciben oportunidades por WhatsApp, formularios, correo u hojas de calculo y necesitan dar mejor seguimiento.
- Equipos que necesitan ordenar reservas, ventas, clientes, eventos, membresias, portales o tareas repetitivas.
- Empresas que aun necesitan una web mas clara, pero tambien pueden requerir software o automatizacion despues del primer lanzamiento.

## Lineas de servicio

- Web estrategica: presencia digital, landing pages, estructura comercial, formularios, WhatsApp y analitica.
- Software a medida: aplicaciones internas, paneles, portales y flujos adaptados a la operacion.
- Automatizacion e integraciones: recordatorios, notificaciones, conexion entre herramientas y seguimiento comercial.
- Analitica y optimizacion: eventos, conversiones, reportes y prioridades de mejora.

## Casos de uso publicados

- Reservas y citas.
- Entradas y eventos.
- Ventas y clientes.
- Academias y membresias.
- Portal de clientes.
- Automatizacion.

Estos textos se redactan como soluciones que NexSystems puede desarrollar, no como productos terminados ni como casos ya vendidos.

## Estructura implementada

La landing queda compuesta asi:

1. Hero.
2. Problem.
3. Services.
4. Solutions.
5. Process.
6. Showcase.
7. LeadMagnet.

## Mensajes clave

- No se trata solo de tener una pagina web, sino de conectar presencia digital, operacion y seguimiento.
- La tecnologia debe ayudar a captar, vender, reservar, atender, ordenar o automatizar.
- Cada solucion se define segun la operacion real de la empresa.
- No se prometen clientes, metricas, certificaciones ni productos terminados.

## Decisiones

- Dominio canonico: `https://www.nexsystems.org`.
- Se mantiene `npm` y `package-lock.json`.
- Se conserva la identidad visual existente; no se hizo rediseño visual amplio.
- Se mantiene GSAP como sistema de animacion y se retira Framer Motion al no quedar imports.
- Los componentes no montados utiles se conservan, pero sin depender de Framer Motion.
- La correccion SSR completa de `<html lang>` para `/en` queda como riesgo tecnico separado.
